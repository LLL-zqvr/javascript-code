### 什么是 jQuery ？
答：<br>
- jQuery 是一个 JavaScript 函数库。
- jQuery 是一个轻量级的"写的少，做的多"的 JavaScript 库。
- jQuery 库包含以下功能：
  1. HTML 元素选取
  2. HTML 元素操作
  3. CSS 操作
  4. HTML 事件函数
  5. JavaScript 特效和动画
  6. HTML DOM 遍历和修改
  7. AJAX
  8. Utilities
### 通过CDN（内容分发网络）引用jQuery
- 百度 CDN:
    <head>
        <script src="https://apps.bdimg.com/libs/jquery/2.1.4/jquery.min.js">
        </script>
    </head>
